# SwiftUI Product Inventory App

A mobile inventory management application built with SwiftUI and Core Data for iOS.  
The application allows users to manage products, search product information, navigate between products, and store data locally using Core Data persistence.

---

## Features

- Display product information
- Navigate through saved products
- Add new products
- Search products by name or description
- View full product list
- Local data persistence using Core Data
- Clean and simple iOS user interface
- Multi-screen SwiftUI navigation

## Installation & Setup

1. Clone the repository

```bash

```
2. Open the project in Xcode
```bash
Open the .xcodeproj file
```
3. Run the application
```bash
Select iPhone Simulator → Click Run
```
